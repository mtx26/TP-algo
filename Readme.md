# TP-algo - Travaux Pratiques d'Informatique

Ce dépôt contient les travaux pratiques d'informatique réalisés dans le cadre du Bac 1 à l'UMONS. Les exercices portent principalement sur l'algorithmique et la programmation.

## Structure du projet

- `TP 1/` : Premier TP, exercices d'introduction à l'algorithmique.
- `TP 2/` : Deuxième TP, approfondissement des notions et résolution de problèmes plus complexes.
- Autres dossiers : Chaque dossier correspond à un TP ou à une série d'exercices.

## Prérequis

- Un éditeur de code (ex: VSCode)
- Python ou le langage utilisé dans les TP
- Git pour la gestion du dépôt

## Utilisation

1. Clonez le dépôt :
   ```bash
   git clone <url-du-repo>
   ```
2. Ouvrez le dossier correspondant au TP souhaité.
3. Suivez les instructions dans chaque fichier pour exécuter les exercices.
